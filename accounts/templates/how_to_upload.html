How to upload files Successfully


To Successfully upload to the family news database you need an audio file and equivalent transcription file. Below we shall guide you on how to successfully create these two items

Audio File
The audio file can be downloaded from WhatsApp, or user can record using phone. The file can be of any format that is mp3, mp4, ogg as long as it is an audio or video file

Transcription
The user has to provide transcription of the audio file. Upload the audio file to any speech to text or audio transcription sites on the internet. There are many out there, common examples include Otter, Transkriptor. Or any other you may prefer. Alternatively you could use dictate function available in Microsft word to transcribe your audio file. You can learn more about this approach from Here. After selecting your preferred transcription method and successfully transcribing the audio file. Save the transcription into a word document, text file or PDF.

Upload Now
After going through the above steps you are now ready to upload using form below.

  • Use the dropdown menu to select name corresponding to your username. Your username is shown at the top of this page below the navigation bar
  • Upload the audio file you created to the "Audio file" Field
  • Upload the word document, or text file or PDF to the "Transcription File" field
  • Double check your information and then click Submit.
